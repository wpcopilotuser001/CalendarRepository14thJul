# CalendarRepository14thJul

A simple Python Flask calendar app that displays today's events.

## Features

- Shows the current weekday and date.
- Lists events scheduled for today.
- Shows a friendly empty state when no events exist for the day.

## Requirements

- Python 3.10+

## Setup

1. Create and activate a virtual environment.
2. Install dependencies:

	pip install -r requirements.txt

## Run

python app.py

Then open http://127.0.0.1:5000 in your browser.
